matchx is a single-instrument CLOB (Central Limit Order Book) matching engine with limit and market orders, price-time and pro-rata algorithms.

<LiveExample />

## Install

```bash
npm install @sargonpiraev/matchx
```

## Features

- Limit and market orders
- Price-Time and Pro-Rata matching
- Partial fills
- Order validation via class-validator
